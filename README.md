# IMDble
A Wordle-inspired game where players get 7 guesses to figure out a random movie based on clues (Actors, Director, Year of Release, etc.) from its IMDb page.

Utilizing the [OMDb](https://www.omdbapi.com) (Open Movie Database) API to fetch IMDb movie data
